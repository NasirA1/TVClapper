
Hardware
========

1. Raspberry Pi Board (Model B - with two USB ports)
2. IguanaWorks Infrared USB Transceiver 
3. USB Microphone (Linux compatible)
4. A modern TV set (with infared remote control capability)


Software
========

1. A Raspberry Pi compatible Linux OS (Wheezy will do just fine!)
2. IguanaWorks IR Transceiver driver (Available from IguanaWorks website)
2. "claptvctrl" application
